# Automated Breast Density Quantification and Breast Mass Analysis in 3D Reconstructed Mammograms (DBT)

This project implements a computational pipeline for analyzing Digital Breast Tomosynthesis (DBT) volumes to quantify volumetric breast density (VBD) and segment suspicious dense masses.

## Project Structure

- `data/`: Placeholder for your datasets (VICTRE, iMerit/Segmed, Hospital dataset).
  - `raw/`: Place your raw DICOM files here.
  - `processed/`: NIfTI volumes and masks generated by the pipeline.
- `src/`: Core source code.
  - `preprocessing/`: DICOM loading, normalization, noise reduction, and breast masking.
  - `density/`: Volumetric breast density estimation and heatmap generation.
  - `segmentation/`: 2.5D U-Net implementation for mass segmentation.
  - `radiomics/`: Radiomic feature extraction.
- `models/`: Trained models and weights.
- `notebooks/`: Jupyter notebooks for exploratory analysis and visualization.
- `scripts/`: Training and validation scripts.

## Installation

1.  Create a virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate  # On Windows: venv\Scripts\activate
    ```
2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Usage

To process a single DBT volume:
```bash
python run_pipeline.py --input path/to/dbt_dicom.dcm --output data/processed/case_01
```

## Next Steps

1.  **Data Loading:** Place your DBT DICOM files into `data/raw/`.
2.  **Dataset Preparation:** Use `src/preprocessing/` to convert and normalize your volumes.
3.  **Training:** Once ground truth masks are available, use `src/segmentation/` to train the 2.5D U-Net.
4.  **Radiomics:** Extract features from the segmented regions for classification tasks.
